## 📌 What is RIPCE?

RIPCE is a fully automated AI-powered conference intelligence platform built for **IAASSE (International Association for Advancement in Science, Sustainability and Education)**.

It transforms the traditional conference registration experience from:

```
Register → Pay → Attend → Certificate
```

Into an intelligent, personalized journey:

```
Register → AI Profile Analysis → Similarity Matching → 
Recommendations (Speakers, Sessions, Papers, Networking) → 
Personalized HTML Report → Email Delivery → Analytics
```

Every researcher who registers receives a **personalized Research Intelligence Report** generated by Claude AI — delivered automatically to their inbox within minutes.

**Nodes:** 10

| Step | Node | Action |
|------|------|--------|
| 1 | Webhook | Receives POST registration payload |

## 📋 Registration API

### Endpoint
```
POST https://raunak420.app.n8n.cloud/webhook/ripce-register
Content-Type: application/json
```

### Request Body

```json
{
  "name": "Dr. Raunak Sharma",
  "email": "raunak@iaasse.org",
  "organization": "YCCE Nagpur",
  "country": "India",
  "research_domain": "Artificial Intelligence",
  "research_interests": "Deep Learning, NLP, Computer Vision",
  "abstract": "This paper explores transformer architectures for multilingual NLP tasks in low-resource settings.",
  "paper_title": "Multilingual Transformers for Low-Resource NLP",
  "keywords": "transformers, NLP, multilingual, low-resource",
  "google_scholar_url": "https://scholar.google.com/citations?user=abc123",
  "linkedin_url": "https://linkedin.com/in/raunaksharma",
  "orcid": "0000-0001-2345-6789",
  "years_of_experience": 8
}
```

### Required Fields
`name` · `email` · `research_domain` · `abstract`

### Success Response (201)
```json
{
  "success": true,
  "researcher_id": "RIPCE-20260623-A1B2C3",
  "message": "Registration successful! Your personalized AI report will be emailed to raunak@iaasse.org shortly."
}
```

### Error Response (400)
```json
{
  "success": false,
  "message": "Missing required fields: name, email, research_domain, abstract."
}
```

## ⚙️ Environment & Setup

### MongoDB Atlas
1. Create cluster at [cloud.mongodb.com](https://cloud.mongodb.com)
2. Create database: `ripce`
3. Get connection string: `mongodb+srv://<user>:<pass>@<cluster>.mongodb.net/ripce`
4. In n8n → Settings → Credentials → New → MongoDB → name it `MongoDB Atlas - RIPCE`

### Anthropic API
1. Get API key at [console.anthropic.com](https://console.anthropic.com)
2. In n8n → Credentials → New → Anthropic → name it `Anthropic - RIPCE`

### Google Sheets
1. In n8n → Credentials → New → Google Sheets OAuth2 → name it `Google Sheets - RIPCE`
2. Create a Google Sheet with two tabs: `Registrations` and `Reports`
3. Paste the Sheet ID into the Google Sheets nodes in WF1 and WF4

### Gmail SMTP
1. Enable 2FA on Gmail → Generate App Password
2. In n8n → Credentials → New → SMTP
   - Host: `smtp.gmail.com`
   - Port: `465`
   - User: your Gmail address
   - Password: App Password
3. Name it `SMTP - RIPCE Email`
